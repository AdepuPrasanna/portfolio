MRECW Results Portal
A production-deployed academic analytics platform serving students of Malla Reddy Engineering College for Women. Students can instantly access complete semester results, backlog reports, class rankings, credits analysis, and SGPA/CGPA growth charts — all without creating an account or logging in. Built on a cache-first Firebase Firestore architecture with a Playwright-powered scraping backend.

Cache-first architecture
Firebase Firestore serves cached results instantly on a cache hit. On a cache miss, Playwright scrapes the official MRECW portal, saves data to Firestore, and returns it. A background refresh keeps cached data current without blocking the user.
Rich academic analytics
Per-semester SGPA and cumulative CGPA growth charts built with Recharts, section-wise class rankings seeded from a sample hall ticket, backlog subject tracking, and a credits analyzer showing earned vs. required progress.
Multi-stage Docker deploy
A single Dockerfile handles a React build stage and a Python/Playwright runtime stage. The final image runs on Render with Gunicorn while Vercel serves the static frontend via CDN — zero infrastructure overlap.
95+ Lighthouse performance
Lazy-loaded routes, vendor/charts/motion code splitting, React Query stale-while-revalidate caching, skeleton loaders, and SSE streaming for class-wide batch fetches — targeting 100 SEO, 100 Accessibility, and 95+ Performance scores.
Key highlights
  • Hall ticket search with persisted local history, autocomplete suggestions, and a side-by-side result comparison view for any two students
  • Dark / light mode with persisted user preference; collapsible sidebar for desktop and bottom navigation bar for mobile
  • No login required at any point — privacy-first design using only hall ticket number for data retrieval
  • SSE streaming endpoint enables real-time progress updates during class-wide result batch fetching across many roll numbers
